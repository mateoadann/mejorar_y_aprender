MÓDULO 03 · Vercel + AI

Vercel se volvió la AI Cloud

Este módulo es el corazón del curso para vos. Vercel armó un stack completo para apps de IA — desde generar UIs con un prompt hasta correr agentes con estado y memoria. Si tu negocio es vender automatizaciones, acá está la caja de herramientas.

Por qué este módulo te importa

Las apps con IA tienen necesidades que una app web normal no tiene: streaming letra por letra, múltiples proveedores de LLM (OpenAI, Anthropic, Google), tiempos de respuesta muy variables (a veces 200ms, a veces 30 segundos), y observability de tokens para no fundirte en costos.

Vercel detectó todos esos problemas y se posicionó como el lugar donde ya están resueltos. No es que inventaron la IA — armaron la infra específica para construir apps de IA en producción. Esa es la tesis del módulo.

El ecosistema AI de Vercel en un vistazo

Son cinco piezas que se combinan. Abajo va el mapa — cada una resuelve una parte del problema, y todas corren sobre la misma foundation (Fluid Compute, que no te cobra el tiempo que el LLM se toma para pensar).

TOOLKIT
AI SDK
Librería TypeScript para hablar con cualquier LLM. Unifica providers.
PROXY
AI Gateway
Una sola API key, routing entre modelos, fallbacks y caching.
UI
v0
Generador de componentes React con prompt. Tailwind + Shadcn listo.
RUNTIME
Vercel Agent
Producto gestionado para correr agentes con estado y memoria.
EXECUCIÓN
Sandbox
Entorno aislado para ejecutar código que generó un LLM.
FOUNDATION
Fluid Compute
La base que hace todo esto viable en serverless — no cobra tiempo de espera a LLMs.
Mateo — conexión directa con tu negocio

Vos ya conocés los Managed Agents de Anthropic del otro curso. Este módulo es el equivalente del lado de Vercel: más orientado al stack web, más integrado con Next.js y React. Entender las dos ofertas te da criterio para decidir qué le proponés a cada cliente.

Las cuatro páginas del módulo

01 · AI SDK

El toolkit TypeScript

Librería open source para hablar con cualquier LLM — OpenAI, Anthropic, Google, Groq — detrás de una sola API. La base sobre la que se construye todo lo demás del ecosistema.

 02 · AI Gateway

La API unificada

Un proxy con una sola API key para 100+ proveedores. Routing entre modelos, fallbacks automáticos, observability central y caching. Cambiás de modelo por configuración, sin tocar código.

 03 · v0 — UI con IA

Componentes generados con prompt

Escribís lo que querés en lenguaje natural y v0 te devuelve componentes React con Tailwind y Shadcn UI. Ideal para prototipos, landings y explorar ideas rápido.

 04 · Agent + Sandbox

Agentes gestionados y ejecución segura

Vercel Agent para correr agentes con memoria de larga duración, Sandbox para ejecutar código generado por IA de forma aislada. Las dos piezas nuevas que cierran el loop del stack.
