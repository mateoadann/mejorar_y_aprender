Lección 04 · 10 min

5 agentes prácticos

Cinco casos reales que mostramos como cards. Cada uno con qué hace, qué tools usa, modo de permiso, costo estimado y un mini diagrama de flujo. Usalos como plantilla para el tuyo.

Sobre el pricing

Managed Agents cuesta $0.08 por session-hour, más los tokens consumidos por el modelo, más $10 por cada 1.000 web searches si usás esa tool. Los costos de abajo son estimaciones mensuales asumiendo uso moderado.

1. Meeting Prep Agent

Principiante

Te arma un briefing antes de cada reunión: historia con el cliente, últimos tickets, comunicación reciente, y puntos abiertos.

Calendar MCP Web search Email read
trigger 30 min antes → pull meeting → research → brief en inbox
Permisos auto-run
Costo ~$15/mes
Dificultad Baja

2. Client Intake Agent

Intermedio

Onboardea clientes nuevos: les hace la primera llamada por email, recolecta información, crea su ficha en el CRM y prepara el kick-off.

Email CRM API Forms
lead entra → draft bienvenida → approval humano → ficha CRM
Permisos mixto
Costo ~$40/mes
Dificultad Media

3. Content Research Agent

Intermedio

Monitorea fuentes específicas (blogs, papers, GitHub trending, HN), clasifica lo relevante y te manda un digest diario con resúmenes.

Web search RSS Email
cron 8am → scan fuentes → filtro relevancia → digest
Permisos auto-run
Costo ~$60/mes
Dificultad Media

4. Document Processing Agent

Avanzado

Extracción estructurada de PDFs y contratos: levanta fechas, montos, cláusulas y los carga en una base de datos. Útil para legal y compras.

PDF parse File IO DB write
upload → parse → validar → persistir
Permisos mixto
Costo ~$80/mes
Dificultad Alta

5. Code Review Agent

Avanzado

Review automático de pull requests: corre tests, revisa diffs, sugiere mejoras y deja comentarios inline. No mergea nunca sin aprobación.

GitHub MCP Shell Linter
PR abierto → checkout → review → comentarios
Permisos mixto
Costo ~$100/mes
Dificultad Alta
Orden recomendado

Si es tu primer agente productivo, empezá por el Meeting Prep: es el más simple, el retorno es inmediato y te obliga a familiarizarte con el ciclo completo sin riesgo.
