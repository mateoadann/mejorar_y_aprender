# Mejorar y Aprender

Repositorio personal de proyectos de aprendizaje sobre IA, automatizaciones y desarrollo de software.

Cada proyecto vive en su propia carpeta con el formato `NNN-slug-del-proyecto`.

## Proyectos

| # | Proyecto | Descripcion |
|---|----------|-------------|
| 001 | [Curso AI Agents](./001-curso-ai-agents/) | Curso visual basado en 3 articulos sobre Claude Managed Agents, memoria de agentes y roadmap AI Engineer 2026. 4 modulos, 21 paginas HTML. |
| 002 | [UI Components Atlas](./002-ui-components-atlas/) | Atlas enciclopedico de 80 componentes UI con demos interactivas, anatomia Shadcn/Radix, vocabulario cross-framework, y prompts para coding agents. 9 modulos, ~95 paginas HTML. |

| 004 | [Agent Infrastructure 2026](./004-agent-infrastructure-2026/) | Curso integrador sobre infraestructura de agentes AI: knowledge layer (KBL + Brand Foundation), stack tecnico con harness thin y progressive disclosure, memoria de 4 capas con dream cycle, skills con self-rewrite, protocols y hooks, optimizacion de knowledge graphs, y operacion real con Claude Code (context rot, rewind, subagents). 8 modulos, 32 paginas HTML. |

## Como agregar un nuevo proyecto

1. Crear carpeta con el siguiente numero disponible: `005-nombre-del-proyecto/`
2. Agregar una fila a la tabla de arriba
3. Cada proyecto es independiente (su propio HTML/CSS/JS o lo que corresponda)

## Stack

- HTML / CSS / JavaScript vanilla
- Tailwind CSS via CDN (en el atlas)
- Sin frameworks, sin dependencias de npm
- Dark mode por defecto
