Stat Card

Stat Metric card KPI card Dashboard tile

Card especializada para mostrar UNA métrica con su trend. Label chico arriba, número grande abajo, porcentaje de cambio al pie. Es el componente rey de los dashboards. Shadcn no lo tiene como componente oficial pero es una composición trivial sobre Card.

Anatomia

StatCard (Root — extiende Card)
Label (small)
[Icon]
Value (big number)
Trend (arrow + % + period)
  • Root Card con padding 6.
  • Label Nombre de la métrica. Small + muted-foreground.
  • Icon Opcional, a la derecha del label.
  • Value El número. Grande (text-3xl) + bold.
  • Trend Arrow up/down + percentage + "vs last month".

Variantes

  • with trend Porcentaje + flecha de dirección.
  • with sparkline Mini-chart al fondo o lateral.
  • with icon Ícono temático en la esquina.
  • with comparison "vs last month" / "vs Q1".
  • colored accent Border-left de color para categorizar.

Cuando usar y cuando no

  • Usalo cuando
  • Dashboards y analytics.
  • KPIs con comparación temporal.
  • Resúmenes ejecutivos.
  • Cards de overview al top de una página.
  • NO lo uses cuando
  • Es texto narrativo: párrafo normal.
  • Son varios datapoints: usá Chart.
  • Es info general no numérica: usá Card.

Prompts para coding agents

Revenue stat card with trend

Build a Stat Card: small muted label "Total Revenue", big number "$45,231.89" (text-3xl font-bold), and a trend line below with an up arrow icon and "+20.1% from last month" in green-500. Use shadcn Card with p-6.

Dashboard grid of 4 KPIs

Create a responsive grid (1 col mobile, 2 tablet, 4 desktop) of 4 StatCards: Revenue, Users, Conversion, Avg Session. Each with label, icon, big value, and colored trend (green positive, red negative).

Stat card with sparkline

Build a StatCard that includes a mini sparkline chart (use recharts LineChart, width 100, height 30, no axes) below the main value. Sparkline color matches trend direction.

Stat card with period comparison

Create a StatCard showing current value, previous period value (smaller, muted), and the delta as a colored chip ("+$1,234 vs last week"). Use lucide TrendingUp / TrendingDown icons.
