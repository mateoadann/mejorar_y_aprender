Variantes

  • default Con header, content y footer.
  • with image Cover image arriba, contenido abajo (product card).
  • stat card Número grande + label + trend. Ver Stat Card.
  • profile card Avatar + datos + acción (follow).
  • outline-only Sin shadow, solo border.
  • elevated Con shadow pronunciado, sin border.

Cuando usar y cuando no

  • Usalo cuando
  • Agrupar información relacionada en una unidad visual.
  • Listas de items visuales (productos, posts, users).
  • Dashboards con KPIs o widgets.
  • Necesitás jerarquía visual clara entre secciones.
  • NO lo uses cuando
  • Tenés datos tabulares: usá Table.
  • Es texto plano narrativo: no necesita contenedor.
  • Es una métrica específica: usá Stat Card.
  • Es una tarea de board: usá Kanban Card.
