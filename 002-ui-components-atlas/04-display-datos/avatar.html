Anatomia

Radix Avatar: Image + Fallback.

Avatar.Root (circular container)
Avatar.Image (img con onError)
Avatar.Fallback (shown if Image fails)
  • Root Contenedor redondo con overflow-hidden.
  • Image Intenta cargar la imagen. Si falla, se esconde.
  • Fallback Aparece cuando Image no existe o falla. Suele mostrar iniciales o ícono.
  • Badge / Status (Opcional) Dot absoluto en la esquina para status online/offline.

Variantes

  • image Imagen real del usuario.
  • initials Iniciales del nombre con bg colorido.
  • icon fallback Ícono genérico cuando no hay nada más.
  • with status dot Indicador online/offline/busy.
  • with badge Notification count en la esquina.
  • group Stack de avatars con overflow ("+3 more").

Cuando usar y cuando no

  • Usalo cuando
  • Representar usuarios en comments, posts, headers.
  • Listas de miembros de un equipo.
  • Author info en cards o artículos.
  • Chat messages con sender.
  • NO lo uses cuando
  • Representar cosas/conceptos: usá Badge o ícono.
  • Es una marca/logo: logo dedicado.
  • No hay persona real (empty state): ícono placeholder.

Prompts para coding agents

Avatar with fallback

Build a shadcn Avatar with AvatarImage (src from user.photoUrl) and AvatarFallback showing the user's initials. Size md (h-10 w-10). Handle onError gracefully.

Avatar group with overflow

Create an Avatar group stack showing up to 4 members with -space-x-2 overlap and ring-2 ring-background. If there are more than 4, show a "+N" avatar at the end with the count.

Avatar with status dot

Build an Avatar with a colored status indicator dot in the bottom-right corner (green=online, yellow=away, red=busy, gray=offline). Use absolute positioning with ring-2 ring-background.

Avatar sizes

Create an Avatar component with 5 size variants (xs=24, sm=32, md=40, lg=48, xl=64). Scale the font-size of the fallback text proportionally.
