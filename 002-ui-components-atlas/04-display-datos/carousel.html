Carousel

Carousel Slider Slideshow Gallery

Slider horizontal que avanza entre slides con botones prev/next, autoplay o swipe. Shadcn usa Embla Carousel debajo. Útil para galerías, testimonials, featured products. Cuidado: si la info es importante, no la escondas detrás de slides — el usuario nunca clickea los prev/next.

  • Root Contenedor con overflow-hidden y estado del índice.
  • Content Track que se desplaza con translateX.
  • Item Cada slide. min-width: 100% usualmente.
  • Previous/Next Botones de navegación.
  • Dots (Opcional) Indicadores clickeables de slide actual.

Variantes

  • horizontal El default. Slides en fila.
  • vertical Slides apilados verticalmente.
  • multi-slide Mostrar 2-3 slides a la vez.
  • autoplay Avanza solo cada N segundos.
  • infinite loop Después del último vuelve al primero.
  • with thumbnails Grid de miniaturas debajo.

Cuando usar y cuando no

  • Usalo cuando
  • Galerías de imágenes (product, fotos).
  • Testimonials rotativos.
  • Featured products o hero slides.
  • Onboarding con pasos visuales.
  • NO lo uses cuando
  • El contenido es crítico (ROI de carousels hero es pésimo).
  • Son vistas alternativas: Tabs.
  • Son pocos items (2-3): grid estático.
  • Es navegación principal.

Prompts para coding agents

Image gallery carousel

Build a shadcn Carousel with 6 product images. Include CarouselPrevious, CarouselNext, and dot indicators below. Aspect ratio 16/9, rounded corners. Use Embla opts with loop: true.

Autoplay testimonials

Create a shadcn Carousel with 5 testimonials (quote, name, avatar, role). Use Embla autoplay plugin at 5000ms. Pause on hover. No prev/next buttons, only dots.

Multi-slide product carousel

Build a Carousel that shows 3 product cards per view on desktop, 2 on tablet, 1 on mobile. Use CarouselItem with basis-1/3 md:basis-1/2 sm:basis-full. 8 products total.

Carousel with thumbnails

Create a product image Carousel with a thumbnail strip below. Clicking a thumbnail navigates to that slide. Sync the active thumbnail with the current slide via Embla's api.selectedScrollSnap().
