Kanban Card

Task card Issue card Trello card Linear issue

Card especializada para representar tareas en boards kanban (Trello, Jira, Linear, Notion). No es componente oficial de ninguna library estándar — es un patrón de composición sobre Card. Incluye título, descripción corta, priority badge, assignees, due date y checklist progress.

Cuando usar y cuando no

  • Usalo cuando
  • Construís un board kanban (Trello, Jira, Linear).
  • Issue tracker o bug tracker.
  • Project management con sprint boards.
  • Task lists con metadata rica.
  • NO lo uses cuando
  • Lista simple sin metadata: List.
  • Datos tabulares: Table.
  • Métrica: Stat Card.
  • Info general: Card.

Prompts para coding agents

Task card with priority and avatars

Build a Kanban task Card with: title (truncate-2), priority Badge top-right (High red / Medium yellow / Low blue), short description, avatar stack footer, due date with Calendar icon. Use shadcn Card + Badge + Avatar.

Task card with checklist progress

Create a task Card showing subtask progress: "3/5 subtasks" label with a CheckSquare icon, and a thin Progress bar below. Bar fills proportional to completed count. Color changes to green when 100%.

Overdue task styling

Build a task Card that shows an "Overdue" state when dueDate < today: change border to red-500/40, add a red Calendar with AlertCircle icon, and show "Overdue · [date]" in red-400 text in the footer.

Draggable kanban board

Set up a Kanban board with 3 columns (Todo, In Progress, Done) using @dnd-kit/core. Each column is a SortableContext containing task Cards. On drag end, update the task's status in state.
