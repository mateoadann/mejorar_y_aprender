Collapsible

Collapsible Collapse (MUI) Show more Disclosure

Una sola sección colapsable con un trigger. Es la versión minimal del Accordion. Si tenés UN solo toggle ("show more", "advanced settings"), usá Collapsible. Si tenés una lista de toggles, usá Accordion.

Anatomia

Radix Collapsible: trigger + content. Nada más.

Collapsible.Root (open state)
Collapsible.Trigger
Collapsible.Content (animated)
  • Root Mantiene el estado open/closed.
  • Trigger Botón que toggle el estado. Puede tener cualquier contenido.
  • Content Panel que se muestra con animación. Puede envolver cualquier cosa.

Variantes

  • with animation Height transition smooth.
  • with chevron Icono que rota 180deg.
  • show more/less Label que cambia según estado.
  • inline Trigger inline dentro de un párrafo.
  • card-based Card entera es el trigger.

Cuando usar y cuando no

  • Usalo cuando
  • "Show more" de texto largo.
  • Advanced settings toggle.
  • Detalles opcionales en cards.
  • Ocultar filtros extra.
  • NO lo uses cuando
  • Tenés varias secciones: Accordion.
  • El contenido es crítico: visible siempre.
  • Es un Dialog: usá Dialog (módulo 03).

Prompts para coding agents

Show more description

Create a shadcn Collapsible for a product description. Show 2 lines by default, with a "Show more" trigger below that expands to reveal the rest. Label toggles to "Show less" and chevron rotates when open.

Advanced settings toggle

Build a Collapsible labeled "Advanced settings" that hides 4 form fields (timeout, retry count, custom headers, debug mode) by default. CollapsibleTrigger styled as a subtle button.

Filter panel collapsible

Create a Collapsible for a filters sidebar. The trigger row shows "Filters" with a count badge and a chevron. Content contains 3 filter groups (checkbox, slider, select).

Inline spoiler

Build an inline Collapsible that hides spoiler content. Trigger is a subtle underlined link "Reveal spoiler", content expands below with a colored background.
