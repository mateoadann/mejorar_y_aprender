Tree View

TreeView Tree File explorer Hierarchical list

Lista jerárquica con nodos colapsables: carpetas que se abren para mostrar subcarpetas y archivos. Pensá en el explorador de VS Code. Es una List pero con profundidad y expand/collapse. Shadcn no lo tiene oficial — se arma custom o con MUI TreeView.

Cuando usar y cuando no

  • Usalo cuando
  • Estructuras jerárquicas: files, carpetas, categorías.
  • Org charts con varios niveles.
  • Nav con sub-sub-items.
  • Representar árboles DOM, JSON o XML.
  • NO lo uses cuando
  • Lista plana sin jerarquía: List.
  • Navegación principal: Sidebar con groups.
  • Solo 2 niveles: Accordion (Accordion).

Prompts para coding agents

File explorer tree

Build a recursive TreeView for a file explorer. Each node has a Folder or File icon (Lucide), a chevron that rotates on expand, and a label. Track expandedIds in state. Clicking the chevron or folder row toggles expansion.

Tree with checkboxes

Create a TreeView with checkbox on each node. Parent checkbox shows indeterminate state when some but not all children are checked. Selecting a parent selects all descendants.

Async tree

Build a TreeView that loads children asynchronously. On first expand of a node, show a spinner and fetch children from an API. Cache them so second expand is instant.

Keyboard navigation

Add keyboard navigation to TreeView: ArrowDown/Up to move, ArrowRight to expand, ArrowLeft to collapse, Enter to select. Follow WAI-ARIA tree pattern with role="tree" and aria-expanded.
