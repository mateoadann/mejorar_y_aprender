List

List ListView ItemList ul / ol

Lista vertical de items homogéneos con o sin separadores. Si tus items tienen la misma estructura (avatar + título + acción), List es tu amigo. Shadcn no tiene List oficial — se arma con <ul> + divide-y. MUI y Ant sí tienen.

Anatomia

List (ul)
ListItem (li)
[Leading] Avatar/Icon
Content (Title + Desc)
[Trailing] Action
Separator (divide-y)
  • Root <ul> o <ol> con border y divide-y.
  • ListItem Una fila. Flex horizontal con 3 slots.
  • Leading Avatar, icono o checkbox.
  • Content Título + descripción (min-w-0 + truncate).
  • Trailing Action (menu, badge, chevron).

Variantes

  • simple Solo texto en cada item.
  • with avatar Leading avatar + content.
  • with action Trailing button o menu.
  • ordered <ol> con numeración.
  • with checkbox Para selección múltiple.
  • interactive Cada item es clickeable (hover state).

Cuando usar y cuando no

  • Usalo cuando
  • Listas homogéneas: users, mensajes, notificaciones.
  • Feeds verticales.
  • Menús simples.
  • Inbox, drawers, side panels.
  • NO lo uses cuando
  • Tenés múltiples columnas: Table.
  • Items heterogéneos: Card grid.
  • Jerarquía anidada: Tree View.
  • Navegación principal: Sidebar (módulo 02).

Prompts para coding agents

Rich list with avatars

Build a List component with ul + divide-y. Each ListItem has: leading Avatar (size 9), main content (name + email truncated), trailing MoreHorizontal button. Hover state with bg-muted/30.

Ordered numbered list

Create an ordered list of top 5 items. Use ol with list-decimal padding, each li with counter and text. Hover state included.

Notification feed

Build a notification list: each item has an icon (Bell, Star, MessageSquare), message text, timestamp (relative), and an unread dot indicator on the right when unread.

Checkbox list for selection

Create a List where each item has a leading Checkbox. Clicking the item (not just the checkbox) toggles selection. Track selected ids in state and show a count above the list.
