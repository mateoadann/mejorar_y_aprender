Accordion

Accordion Collapse (Ant) Expander Disclosure group

Secciones colapsables donde solo una (single) o varias (multiple) pueden estar abiertas. Clásico FAQ. OJO — Accordion vs Collapsible: Accordion es una lista de items colapsables (piensa en FAQ con 5 preguntas). Collapsible es UN solo item con toggle ("show more"). No los confundas.

Anatomia

Radix Accordion. Estándar cross-framework.

Accordion.Root (type="single" | "multiple")
Accordion.Item (value="a")
Accordion.Trigger (header)
Accordion.Content (panel)
Accordion.Item (value="b")
  • Root Type="single" o "multiple". Mantiene estado global.
  • Item Cada sección. Tiene un value único.
  • Trigger Header clickeable con ícono chevron.
  • Content Panel que se muestra/oculta con animación.

Variantes

  • single Solo una sección abierta a la vez.
  • multiple Varias secciones pueden estar abiertas simultáneamente.
  • controlled Estado manejado desde el padre (prop value/onValueChange).
  • with chevron Icono chevron-down que rota 180deg.
  • with plus/minus Iconos + y − alternando.
  • bordered Cada item con border full (tipo card).

Cuando usar y cuando no

  • Usalo cuando
  • FAQs con varias preguntas.
  • Settings agrupados por sección.
  • Contenido opcional que satura la página si se muestra todo.
  • Product details largos (descripción, specs, reviews).
  • NO lo uses cuando
  • El usuario siempre necesita ver el contenido → display directo.
  • Son vistas alternativas: Tabs (módulo 02).
  • Es un solo item: Collapsible.
  • Jerarquía profunda: Tree View.

Prompts para coding agents

FAQ accordion single mode

Build a shadcn Accordion with type="single" and collapsible. Include 5 FAQ items about a SaaS product. Each AccordionTrigger has a question, AccordionContent has the answer. Chevron rotates 180deg when open.

Settings accordion multiple

Create an Accordion with type="multiple" for a settings page. Sections: Profile, Security, Notifications, Billing, Integrations. Each content has form fields. Multiple can be open at once.

Product details accordion

Build an Accordion for a product page with sections: Description, Specifications (table), Shipping, Reviews. Use shadcn Accordion, first item open by default via defaultValue.

Accordion with plus/minus icons

Create an Accordion where the trigger shows a Plus icon when collapsed and Minus when expanded. Use data-state in the AccordionTrigger to conditionally render. Border between items only.
