Aspect Ratio

AspectRatio aspect-* ratio

Un contenedor que mantiene una proporción fija entre ancho y alto, sin importar cuánto escale. Antes de CSS aspect-ratio, esto se hacía con padding-bottom hack. Hoy es una línea. Y resuelve el problema clásico de videos e imágenes que "saltan" mientras cargan.

Anatomía

Radix AspectRatio: un Root con ratio prop y children libres adentro.

AspectRatio.Root ratio={16/9}
children (img / video / iframe)
  • Root El wrapper con aspect-ratio aplicado.
  • ratio prop Radix recibe un número (16/9 = 1.777). Tailwind usa aspect-video, aspect-square, etc.
  • Children Lo que va adentro. Normalmente img, video, iframe o un placeholder.
  • object-fit Para imágenes: object-cover (recorta) o object-contain (deja barras).

Cuándo usar y cuándo no

  • Usalo cuando
  • Videos (16:9 o 21:9) que deben ser responsive
  • Avatares de usuario (1:1 garantiza el círculo perfecto)
  • Cards de producto con imágenes de proporción conocida
  • Iframes de YouTube / Vimeo / Spotify
  • Placeholders de imagen mientras carga (evita layout shift)
  • NO lo uses cuando
  • El contenido tiene altura variable (un card con texto)
  • Texto narrativo o contenido de lectura
  • Componentes de formulario (inputs, textareas)
  • Ya tenés una imagen con width/height attributes que hacen el mismo trabajo

Prompts para coding agents

16:9 video embed

Create a 16:9 aspect ratio container using shadcn AspectRatio with ratio={16/9}. Inside, embed a YouTube iframe with src from a prop. Apply rounded-lg and overflow-hidden to the container.

Square avatar with image

Create a 1:1 aspect ratio square (aspect-square) containing a user profile image. Apply object-cover to the img so it fills without distortion, rounded-full for circular avatar.

Responsive card gallery with 4:3 images

Create a grid of product cards. Each card has an image with aspect-[4/3], rounded-t-lg, object-cover. Below the image: title, price, CTA button. The aspect ratio prevents layout shift during image load.

Skeleton loader with aspect ratio

Create a loading skeleton for a video card: a div with aspect-video, bg-muted, animate-pulse, rounded-lg. This reserves space before the video loads, preventing CLS.
