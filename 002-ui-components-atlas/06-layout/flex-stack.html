Flex / Stack

Flex Stack HStack VStack Box

Layout lineal. Una sola dimensión: o filas (HStack) o columnas (VStack). El 80% del layout de una app real es esto. Grid es para estructuras 2D. Flex es para toolbars, listas, form rows, headers. Aprendé a distinguirlo y no vas a sufrir más.

Anatomía

Un flex container tiene 4 propiedades fundamentales.

Flex.Root — display: flex + direction + align + justify + gap
child 1
child 2

  • flex-direction row (horizontal) o column (vertical). Es el eje principal.
  • align-items Alineación en el eje CRUZADO. center / start / end / stretch.
  • justify-content Alineación en el eje PRINCIPAL. between / around / evenly / center.
  • gap Espacio entre hijos. Mucho mejor que margin.
  • flex-wrap wrap / nowrap. Si los items no entran, saltan a la siguiente línea.

Cuándo usar y cuándo no

  • Usalo cuando
  • Toolbars, button groups, action bars
  • Headers con logo a la izquierda y nav a la derecha
  • Listas verticales de cards o items
  • Form rows (label + input + helper text)
  • Navegación tab-like
  • NO lo uses cuando
  • Layouts 2D con filas Y columnas → Grid
  • Necesitás posiciones específicas (header arriba, sidebar izq, main) → Grid con areas
  • Solo querés centrar contenido en un max-width → Container
  • Estás armando un formulario complejo con alineación perfecta de columnas → Grid

Prompts para coding agents

Horizontal button toolbar

Create a horizontal toolbar using flex with items-center and gap-3. Include 4 buttons: Save (primary), Cancel (outline), Preview (outline), Share (outline). All buttons h-9 px-3 text-sm.

Vertical stack of cards

Create a VStack (flex flex-col gap-4) containing 3 card components. Each card has a title, description, and action button. The stack should stretch to full width of its container.

Header with logo and nav

Create a sticky header using flex items-center justify-between with px-6 py-4. Left side: logo + brand name (flex items-center gap-2). Right side: nav links + user avatar.

Wrapping chip list

Create a flex container with flex-wrap and gap-2 that holds a list of tag chips. Chips should wrap to the next line when they overflow the container width.
