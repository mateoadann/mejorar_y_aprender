Spacer

Spacer Flex grow flex-1 <div /> vacío

Un elemento vacío que crece para ocupar el espacio disponible. Suena tonto, pero es clave: cuando querés que dos elementos queden en extremos opuestos pero con contenido variable en el medio, no podés usar justify-between. Ahí entra el Spacer.

  • Root Un div (o span) con flex: 1 (flex-1). Sin contenido.
  • Variante fija Un div con altura o ancho fijo (h-8 / w-4) para gaps específicos donde gap no sirve.
  • Key insight En un flex container, flex-1 hace que el elemento se "estire" para llenar el espacio libre.

Spacer vs gap vs margin

Tres formas de crear espacio. Saber cuál usar es crítico.

gap: espacio UNIFORME entre todos los children de un flex/grid. Tu default.
Spacer (flex-1): empuja children a los extremos cuando hay contenido variable en el medio.
margin: legacy. Usalo solo si no controlás el parent (ej: component externo).

Cuándo usar y cuándo no

  • Usalo cuando
  • Querés empujar UN item al extremo mientras el resto queda junto
  • El número de items del medio es variable (no sabés cuántos son)
  • Necesitás un gap específico entre dos grupos dentro del mismo flex
  • Sidebar con logo arriba y user menu abajo (VStack con Spacer en el medio)
  • NO lo uses cuando
  • Espaciado uniforme entre todos los items → usá gap
  • Padding interno → usá padding utility
  • Justify between ya alcanza (solo 2 items en el container)
  • Separación visual → usá Divider o Separator Section

Prompts para coding agents

Toolbar with spacer pushing last button

Create a horizontal toolbar with 3 buttons on the left (File, Edit, View), then a flexible spacer (div with flex-1), then a Logout button pushed to the right. Use flex items-center gap-3 on the container.

Sidebar with spacer between sections

Create a vertical sidebar (flex flex-col h-screen). At top: logo and nav links. Add a Spacer (flex-1). At bottom: user profile card. The spacer pushes the profile to the bottom regardless of nav length.

Fixed spacer between form sections

Inside a vertical stack, create two form sections separated by a fixed spacer of 2rem (h-8). No divider line, just empty space for visual breathing room.

Card footer with ml-auto button

Create a card footer with "Learn more" link on the left and a "Subscribe" button pushed to the right using ml-auto. Use flex items-center on the footer.
