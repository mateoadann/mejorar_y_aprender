Anatomía

Shadcn Resizable: PanelGroup contiene Panels separados por Handles.

ResizablePanelGroup direction="horizontal"
ResizablePanel defaultSize={40}
ResizablePanel defaultSize={60}
  • ResizablePanelGroup El contenedor. Define direction: horizontal o vertical.
  • ResizablePanel Cada panel. Tiene defaultSize, minSize, maxSize (en %).
  • ResizableHandle El divisor draggeable entre paneles. Puede tener withHandle para mostrar grip icon.
  • collapsible Un panel puede colapsarse cuando llega al minSize.

Cuándo usar y cuándo no

  • Usalo cuando
  • Editores tipo VS Code / Cursor con sidebar + main + panel
  • Dashboards con widgets que el user quiere ajustar
  • Side-by-side comparison views (diff viewers, code compare)
  • Desktop apps donde el usuario espera control total del layout
  • NO lo uses cuando
  • Layouts fijos con tamaños conocidos → Grid
  • Mobile (el drag por touch es frustrante)
  • Layouts marketing o landing pages
  • Users no técnicos que no esperan drag interactions
