Anatomía

El grid tiene dos niveles: el contenedor y los items.

Grid.Root — display: grid + grid-template-columns + gap
item 1
item 2
item 3 (col-span-1)
  • Grid.Root El contenedor con display: grid. Define columnas, filas, gap y áreas.
  • grid-template-columns Cómo se distribuyen las columnas: fijas, fr (fraction), minmax, auto-fit.
  • gap Espacio entre items (reemplaza el margin hack de antes).
  • Item Cada hijo directo. Puede hacer col-span-*, row-span-*, col-start-*.
  • Grid areas Nombres para posiciones específicas: "header header" "sidebar main".

Cuándo usar y cuándo no

  • Usalo cuando
  • El layout tiene filas Y columnas (dashboards, galerías de cards)
  • Necesitás posiciones específicas con grid-area
  • Cards que se reacomodan automáticamente según ancho (auto-fit)
  • Formularios con columnas alineadas perfectamente
  • Tables visuales que no son <table>
  • NO lo uses cuando
  • El layout es unidimensional (lista vertical, toolbar horizontal) → Flex
  • Solo necesitás centrar contenido y limitar ancho → Container
  • El orden de los items cambia dinámicamente según el contenido
  • Estás haciendo una tabla de datos real → <table>

Prompts para coding agents

Fixed 12-column grid with spans

Create a 12-column CSS grid with a 1rem gap. The first card spans 8 columns (col-span-8) for the main content, the second card spans 4 columns (col-span-4) as an aside. Below, two cards each spanning 6 columns.

Auto-fit responsive card grid

Create a responsive card grid using CSS Grid with `grid-template-columns: repeat(auto-fit, minmax(280px, 1fr))` and a 1rem gap. Each card should be at least 280px wide and stretch to fill available space.

Named grid areas layout

Create a page layout using CSS grid-template-areas with zones named: "header", "sidebar", "main", "footer". Header and footer span full width, sidebar is 240px on the left, main fills the rest. Use grid-area on each child.

Masonry-like product grid

Create a 4-column grid on desktop, 2 on tablet, 1 on mobile using grid-cols-1 md:grid-cols-2 lg:grid-cols-4. Add gap-6 between items. Each child is a Card component.
