Rich Text Editor

Tiptap Lexical ProseMirror Quill / Slate WYSIWYG

Editor WYSIWYG para texto con formato: bold, italic, headings, lists, links, images. Debajo de todo hay siempre un document model (no es solo HTML). Tiptap (sobre ProseMirror) y Lexical (Meta) son los estándares modernos.

Demo interactiva

Probá el editor. Seleccioná texto y aplicá bold, italic o convertilo en heading.

  • Item uno
  • Item dos

Atajos: Ctrl/Cmd+B = bold, Ctrl/Cmd+I = italic.

Variantes

  • Full toolbar — todos los controles visibles.
  • Minimal toolbar — solo lo esencial.
  • Floating toolbar — aparece on selection.
  • Bubble menu — tooltip-style al seleccionar.
  • Slash commands — "/heading", "/image" (Notion-like).
  • Markdown shortcuts — "## " se convierte en H2.
  • Collaborative — Yjs + Tiptap (multi-usuario).
  • Read-only — display formateado sin edit.

Estados

  • Idle
  • Focused
  • Selection active
  • Formatting applied
  • Empty (placeholder)
  • Read-only

Cuándo usar

Sí, usalo cuando

  • Blog editors, CMS, email composers
  • Notes apps, documentation editors
  • Comments con formato (bold, links)
  • Collaborative docs (Notion-like)

No, evitalo cuando

  • Texto simple sin formato → Textarea
  • Markdown-first → Markdown editor con preview
  • Código → Code Editor (Monaco/CodeMirror)
  • Comentarios cortos → Textarea con autolink
