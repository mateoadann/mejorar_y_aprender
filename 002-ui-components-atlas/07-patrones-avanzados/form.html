Form

Shadcn Form MUI FormControl Ant Form React Hook Form

Formulario completo con validación en vivo, error handling, accessibility y estados de submit. No es un input suelto — es el patrón que junta label + control + description + error message.

Anatomía

Form (provider con schema)
FormField → FormItem
FormLabel
FormControl → Input / Select / Textarea
FormDescription (hint)
FormMessage (error text)
Submit area
Button (loading state)

Variantes

  • Single-column — default, mobile-first.
  • Two-column — para forms largos en desktop.
  • Inline — filters, search bars.
  • Con fieldsets — secciones agrupadas.
  • Validación on blur — valida al salir del campo.
  • Validación on submit — valida al enviar.
  • Con server errors — errores del backend.
  • Optimistic UI — actualiza UI antes del response.

Estados

  • Pristine
  • Dirty
  • Validating
  • Submitting
  • Success
  • Error (campo)
  • Error (servidor)
  • Disabled

Cuándo usar

Sí, usalo cuando

  • Signup, login, edit profile, settings
  • Necesitás validación custom
  • Hay dependencias entre campos

No, evitalo cuando

  • Un solo input → input standalone
  • Búsqueda simple → Search bar
  • Flujo largo secuencial → Form Wizard
