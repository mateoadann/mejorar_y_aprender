Masked Input

iMask.js Cleave.js react-imask use-mask-input

Input con formato forzado mientras el usuario escribe. Teléfonos, tarjetas de crédito, fechas manuales, CUIT. El mask inserta separadores automáticamente y bloquea caracteres inválidos.

Variantes

  • Phone — (+54) ## ####-####
  • Credit card — #### #### #### ####
  • Date — DD/MM/YYYY
  • CUIT/CUIL — ##-########-#
  • Time — HH:MM
  • IP address — ###.###.###.###
  • Currency — $#,###.##
  • Custom regex — patrones complejos.

Cuándo usar

Sí, usalo cuando

  • Teléfonos, tarjetas, documentos (CUIT/DNI)
  • Fechas manuales con formato estricto
  • Códigos con estructura fija (IPs, horarios)
  • Campos donde el error de formato es costoso

No, evitalo cuando

  • Texto libre → Input
  • OTP/PIN → OTP Input (Módulo 01)
  • Fecha con calendario → Date Picker
  • Número sin formato → input type="number"
