Kanban Board

Trello-style Linear board Jira board Task board

Board con columnas por status y cards draggables entre columnas. El combo: Kanban Card (Módulo 04) + Drag & Drop + Columnas. Linear, Jira, Trello — todo el mundo terminó reimplementándolo.

Demo interactiva

Arrastrá cards entre columnas.

Anatomía

Board Root (+ DndContext)
Column[] (droppable zones)
Header: título + count + add button
Scrollable Card List (Kanban Cards draggables)
Footer: add card button
Card (Kanban Card del Módulo 04)
Title + tags + assignee + metadata

Variantes

  • 3 columnas — Todo / Doing / Done.
  • Con swimlanes — grouped by priority/assignee.
  • Con WIP limits — max cards por columna.
  • Con filters — by assignee, tag, priority.
  • Sprint board — agile con burndown.
  • Pipeline CRM — deal stages.
  • Content workflow — draft/review/published.
  • Vertical mobile — columns como tabs en mobile.

Estados

  • Idle
  • Dragging card
  • Drag-over column
  • Card dropping
  • Adding card
  • WIP limit reached

Cuándo usar

Sí, usalo cuando

  • Task management (Linear, Jira, Trello)
  • Pipeline tracking (CRM deals)
  • Content workflows (draft/review/published)
  • Recruitment (candidate stages)

No, evitalo cuando

  • Listas lineales sin status → List
  • Muchos items → Data Table (mejor escalabilidad)
  • Tareas jerárquicas → Tree View
  • Mobile-only (kanban no es mobile-friendly)

Alternativas relacionadas

  • Data Table — para muchas tasks con filtros complejos.
  • Tree View — para tareas jerárquicas.
  • Calendar / Timeline — cuando el tiempo es el eje.
