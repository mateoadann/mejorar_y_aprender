Code Editor

Monaco CodeMirror 6 Ace Editor Shiki

Editor especializado para código con syntax highlighting, line numbers, autocomplete, linting. Monaco es el engine de VS Code (pesado pero feature-complete). CodeMirror 6 es más liviano y modular.

Nota: este demo es un mock visual. Un code editor real requiere librerías como Monaco o CodeMirror — imposible de replicar con JS vanilla inline.

import { api } from './api';

export async function getUser(id: string) {
  const res = await api.get(`/users/${id}`);
  if (!res.ok) {
    throw new Error('User not found');
  }
  return res.json();
}

Variantes

  • Monaco — feature-complete, pesa ~3MB.
  • CodeMirror 6 — modular, más liviano.
  • Solo highlighting — Shiki/Prism, no edición.
  • Diff editor — 2 panes con cambios.
  • Collaborative — OT o CRDT para multi-usuario.
  • Multi-file — con tabs y file tree.
  • Con LSP — Language Server Protocol para intellisense real.
  • Con preview — REPL / runtime en vivo.

Cuándo usar

Sí, usalo cuando

  • Playgrounds (CodeSandbox, StackBlitz)
  • Config editors dentro de apps
  • Tutoriales interactivos
  • IDE features en la web

No, evitalo cuando

  • Texto plano → Textarea
  • Texto con formato → Rich Text Editor
  • Solo mostrar código read-only → Shiki/Prism
  • Snippets cortos inline → <pre><code>
