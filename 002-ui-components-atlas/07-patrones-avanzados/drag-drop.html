Drag & Drop

dnd-kit react-dnd react-beautiful-dnd Sortable

Elementos que el usuario arrastra y suelta para reordenar o mover entre zonas. Kanban boards, file uploads, form builders, gallery arrangement. En el browser tenés HTML5 DnD nativo — pero para algo serio usá dnd-kit.

Demo interactiva (reorder)

Arrastrá los items para reordenarlos.

    Anatomía

    DndContext (provider, maneja estado global del drag)
    Draggable Items
    Item + drag handle
    Droppable Zones
    Contenedor destino
    Drag Overlay
    Preview visual durante el drag
    Collision Detection
    closestCenter, rectIntersection...

    Variantes

    • Sortable list — reorder dentro de una lista.
    • Multi-list move — mover entre zonas (kanban).
    • Grid reorder — reordenar items en grid 2D.
    • File drop — drag archivos del OS.
    • Con drag handle — solo una parte es draggable.
    • Con animations — smooth reorder.
    • Con drop preview — placeholder destino.
    • Multi-select drag — arrastrar varios a la vez.

    Estados

    • Idle
    • Dragging
    • Drag over (zone)
    • Drop accepted
    • Drop rejected
    • Drop cancelled

    Vocabulario cross-framework

    LibreríaParadigmaNotas
    @dnd-kit/coreHooks + componentesModerno, accessible, SSR-ok
    react-dndHOC + hooksLow-level, backend HTML5 o touch
    react-beautiful-dndDeclarativoDeprecated, no usar en nuevo código
    SortableJSImperativo (vanilla)Framework-agnostic
    HTML5 DnD APINativeLimitado, mobile-touch flaky

    Cuándo usar

    Sí, usalo cuando

    • Kanban, reorder lists, form builders
    • File drag & drop uploads
    • Gallery arrangement, image editors
    • Workflow designers (nodos + edges)

    No, evitalo cuando

    • Solo seleccionar → Checkbox
    • Reorder simple → botones up/down
    • Mobile-first sin alternativa — touch DnD es complicado
    • Accesibilidad crítica sin alternativa teclado
