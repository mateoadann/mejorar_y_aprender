Dropdown Menu

DropdownMenu Menu Dropdown ActionsMenu

Menú contextual con items estructurados (item, checkbox, radio, submenu, separator, shortcut) disparado por un trigger button. Es el "..." de toda la UI moderna: abrir el menú del user, mostrar acciones de una row, exponer opciones secundarias. A diferencia de un Popover, los items ya vienen con keyboard navigation, roving tabindex y roles ARIA correctos.

Dropdown Menu vs Popover vs Select: Popover = contenido libre. Select = elegir un valor de un form. Dropdown Menu = lista de acciones estructuradas con keyboard nav built-in.

  • Root Controla open/close state.
  • Trigger El botón que abre el menú (recibe focus y aria).
  • Portal Monta el Content fuera del DOM parent, evita overflow hidden.
  • Content El panel flotante con todos los items.
  • Item Acción normal, cierra el menú al click.
  • CheckboxItem / RadioItem Estado toggleable, NO cierran el menú.
  • Sub Submenu anidado que se abre al hover del SubTrigger.
  • Shortcut Muestra el atajo (cosmético, no lo registra).

Cuándo usar y cuándo no

  • Usalo cuando
  • Acciones contextuales sobre un item (edit, delete, duplicate)
  • Menús "..." de more actions en tables o cards
  • User menu (profile, settings, logout)
  • Necesitás keyboard nav built-in con roles ARIA correctos
  • NO lo uses cuando
  • Selección de valor para un form → Select / Combobox
  • Navegación principal del sitio → Navbar
  • Contenido libre (form, preview) → Popover
  • Right-click en un elemento → Context Menu

Prompts para coding agents

Dropdown menu with items, separator and keyboard shortcuts

Create a shadcn DropdownMenu triggered by an "Options" button. Inside render a Label "My Account", two Items (Profile with ⇧⌘P, Settings with ⌘S), a Separator, a CheckboxItem "Show status bar", another Separator, and a destructive Item "Log out" with ⇧⌘Q shortcut visible on the right.

User menu dropdown with profile, settings and logout

Create a DropdownMenu triggered by the user avatar in the top-right. Inside render the user name and email as a Label, a Separator, Items for Profile, Billing, Team, then a Separator, then a Log out item that calls signOut() on click.

Table row actions dropdown

Add a DropdownMenu triggered by a MoreHorizontal icon button at the end of each table row. Items: View, Edit, Duplicate, Separator, Delete (destructive). Each item calls the correct mutation with the row id.

Dropdown with nested submenu

Create a DropdownMenu with a SubMenu "More tools" that contains Save Page As, Create Shortcut, Name Window, Developer Tools. The submenu should open to the right of the parent on hover.
