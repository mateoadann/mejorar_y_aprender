Context Menu

ContextMenu Right-click menu ContextualMenu

La misma idea que Dropdown Menu pero el trigger es el evento contextmenu (click derecho o long-press en touch). Aparece exactamente donde el usuario clickeó, no anclado a un botón. Es el pattern de file managers, editores de código, canvases: Photoshop, Finder, VS Code, Figma.

Context Menu vs Dropdown Menu: mismo menú visualmente, distinto disparador. Context = click derecho (oculto, avanzado). Dropdown = botón visible (descubrible).

  • Trigger Envuelve el área "right-clickeable". Intercepta el evento contextmenu del browser.
  • Portal + Content El menú se posiciona absolutamente en las coordenadas del click.
  • Items Idénticos a DropdownMenu (Item, CheckboxItem, RadioItem, Separator, Sub).
  • Sin trigger visible El usuario debe saber que puede hacer right-click. Es "oculto".

Cuándo usar y cuándo no

  • Usalo cuando
  • File manager / tree con muchas acciones por ítem
  • Editores de código / texto (Copy, Cut, Paste, Go to definition)
  • Canvases / diagrams (Figma, Miro, Excalidraw)
  • Usuarios técnicos que esperan el pattern de desktop
  • NO lo uses cuando
  • Interfaces mobile (no hay right-click, el long-press compite con scroll)
  • Usuarios no técnicos que no saben que existe el right-click
  • Acciones críticas que NO deben estar ocultas → DropdownMenu visible
  • Es la ÚNICA forma de llegar a una acción (siempre duplicá el acceso)

Prompts para coding agents

Context menu on right-click with copy/paste/delete actions

Wrap each list item in a shadcn ContextMenu. On right-click open a menu with Copy (⌘C), Cut (⌘X), Paste (⌘V), a Separator, and a destructive Delete item. Each action calls the corresponding clipboard or store method with the item id.

File explorer context menu with rename and move options

Add a ContextMenu to each row in the file explorer tree. Items: Open, Open in New Tab, Separator, Rename (F2), Move to… (SubMenu listing target folders), Separator, Delete (destructive). Make sure the menu positions at the cursor location.

Canvas context menu for shapes

On the Figma-like canvas, add a ContextMenu that triggers on right-click of a selected shape. Items: Bring to Front, Send to Back, Separator, Group (⌘G), Ungroup (⇧⌘G), Separator, Delete. Each calls the canvas store action.
