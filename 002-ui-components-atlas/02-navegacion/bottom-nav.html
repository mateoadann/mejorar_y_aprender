Bottom Nav

Bottom Navigation Tab Bar Footer Nav Mobile Tabs

Barra fija en el borde inferior de apps mobile. 3-5 items con icono (+ label opcional) para saltar entre las secciones principales. Es el estandar de iOS (Tab Bar) y Android (Bottom Navigation Bar). Shadcn no la trae oficial, la armas con Tailwind fixed bottom-0. No la confundas con Tabs: bottom nav es navegacion entre PANTALLAS, Tabs son alternativas dentro de una vista.

NavItem (active)
NavItem
  • Root <nav> fixed al bottom con safe-area inset.
  • Item Un link por seccion. 3-5 items maximo.
  • Icon SVG outline en inactive, filled en active.
  • Label Texto corto (10px). A veces se omite en Android.
  • Badge Contador de notificaciones (opcional).
  • Active indicator Color, underline, o pill de fondo.
  • FAB (Material) boton elevado al centro.

Variantes

  • icon-only Solo iconos, sin labels (Twitter X mobile).
  • icon + label El default Material.
  • with-fab Boton de accion principal elevado al centro.
  • pill-active Active con pill de fondo (Material 3).
  • ios-style Icon + label chico, active con color tint.

Cuando usar y cuando no

  • Usalo cuando
  • App mobile con 3-5 secciones principales.
  • Necesitas acceso rapido siempre visible.
  • Las secciones son de igual jerarquia.
  • NO lo uses cuando
  • Desktop: usa Navbar o Sidebar.
  • Mas de 5 secciones: usa Drawer o Sidebar.
  • App de un solo flujo: no necesitas nav persistente.
  • Contenido dentro de una pantalla: usa Tabs.

Prompts para coding agents

Mobile bottom nav with 5 icons and active indicator

Create a mobile bottom navigation bar with 5 items: Home, Search, Add (center, elevated as FAB), Notifications, Profile. Use lucide icons. Active item has filled icon and bold label. Fixed to bottom with border-top and backdrop-blur. Only visible below md breakpoint.

Bottom nav with notification badge

Build a bottom nav where the Notifications item shows a red badge with the unread count (from a prop). Badge is positioned top-right of the icon. Use Tailwind.

Bottom nav synced with React Router

Create a bottom nav where each item is a NavLink from react-router-dom. Use useLocation to determine the active item by matching pathname startsWith the item's "to" prop.

Bottom nav with safe-area for iOS

Build a bottom navigation bar that respects iOS safe-area-inset-bottom using pb-[env(safe-area-inset-bottom)]. The bar itself should be sticky at the bottom with a solid background.
