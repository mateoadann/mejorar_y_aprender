Stepper

Stepper Steps Wizard Progress Steps

Indicador visual de pasos secuenciales. Muestra donde estas, que hiciste y que falta. El uso canonico: checkout, onboarding, formularios multi-step. Importante: Stepper NO es Tabs. Tabs son alternativas paralelas; Stepper implica orden y progresion. Shadcn no tiene oficial, lo armas con Tailwind.

Anatomia

Root (ol) con Steps y Connectors entre ellos.

Stepper (ol)
Step (li)
└ StepIndicator (number | check | error)
└ StepLabel
└ StepDescription (optional)
StepConnector (line between steps)
  • Stepper Root <ol>. Orientacion horizontal o vertical.
  • Step Un paso. Contiene indicator + label.
  • StepIndicator Circulo con numero, check (completado) o X (error).
  • StepLabel Titulo del paso.
  • StepDescription Subtitulo opcional.
  • StepConnector Linea entre steps. Color cambia segun progreso.

Variantes

  • horizontal Steps en fila. El mas comun para desktop.
  • vertical Steps apilados. Mobile o wizards con descripcion larga.
  • with-icons Reemplaza el numero por iconos semanticos (cart, credit-card).
  • linear No podes saltar pasos. Tenes que completar en orden.
  • non-linear Podes clickear cualquier step para saltar.

Cuando usar y cuando no

  • Usalo cuando
  • Checkout de ecommerce.
  • Onboarding de usuario nuevo.
  • Formulario largo dividido por secciones.
  • Proceso con pasos obligatorios en orden.
  • NO lo uses cuando
  • Contenido paralelo sin orden: usa Tabs.
  • Navegacion entre paginas: usa Breadcrumbs.
  • Lista de resultados: usa Pagination.

Prompts para coding agents

Horizontal stepper for checkout with 4 steps

Create a horizontal checkout stepper with 4 steps: Cart, Shipping, Payment, Review. Each step has a number circle, label, description. Completed steps show a check icon. Current step has a filled ring. Use Tailwind, no library.

Vertical stepper with error state

Build a vertical stepper with 5 steps for a KYC onboarding flow. Step 3 should display in error state (red circle with X icon) and show an error message below its label.

Stepper controlled with Next/Back buttons

Create a linear stepper with React useState for current step. Below the stepper, show the content of the active step and Next/Back buttons. Disable Back on step 0 and Next on the last step.

Stepper with icons instead of numbers

Build a stepper where each step uses a lucide icon instead of a number: ShoppingCart, Truck, CreditCard, CheckCircle. Keep the completed state showing a check overlay.
