Navbar

Navbar AppBar Header Top Navigation Menu mode="horizontal"

Barra horizontal en el tope de la pagina con logo, links de navegacion y acciones (search, user menu, CTA). No es un componente primitivo: casi ningun design system lo exporta como tal. Es un patron que armas componiendo un <nav> + links + Button + Avatar. Por eso cuando le pedis "un navbar" al agente tenes que ser especifico con la anatomia.

Anatomia

Tres zonas: Brand, Links, Actions. Flex, gap, justify.

nav.Root (container)

Actions
Search · Button · UserMenu
  • Root Elemento <nav> semantico. Semantica importa.
  • Brand Logo + nombre. Link al home.
  • NavLinks Lista de links principales. Marca el activo con estado visual.
  • Actions Todo lo que no es navegacion: search, notificaciones, avatar, CTA.
  • MobileMenuTrigger En mobile, hamburguesa que abre un Drawer con los links.

Variantes

Nombres estandar para pedirlas al agente.

  • sticky Se queda pegado arriba al scrollear (sticky top-0).
  • transparent Sin bg al inicio, solido despues de scroll X px (landing pages).
  • with-search Input de busqueda centrado o a la derecha.
  • with-user-menu Avatar dropdown con profile, settings, logout.
  • centered-logo Logo centrado, links a los costados (estilo e-commerce).

Cuando usar y cuando no

  • Usalo cuando
  • Paginas publicas, marketing, landing.
  • Apps desktop-first con pocas secciones (3-7).
  • Necesitas branding visible siempre.
  • Queres search global disponible en toda la app.
  • NO lo uses cuando
  • Tenes 15+ secciones jerarquicas: usa Sidebar.
  • App mobile-first con 3-5 tabs: usa Bottom Nav.
  • Dashboard complejo con sub-niveles: Sidebar es mejor.

Prompts para coding agents

Sticky navbar with search and user menu

Create a sticky top navbar with: logo on the left, 4 nav links in the center (Dashboard, Projects, Team, Docs), a search input and user avatar dropdown on the right. Use Tailwind, dark theme, backdrop-blur on scroll. Mark active link with bg-accent.

Transparent navbar that becomes solid on scroll

Build a transparent navbar for a landing page. When the user scrolls past 80px, add a solid background and subtle border-bottom. Use React state + scroll listener. Tailwind classes only.

Responsive navbar with mobile drawer

Create a navbar that on mobile collapses nav links into a hamburger button that opens a shadcn Sheet from the right containing the same links vertically. Desktop shows all links inline.

Navbar with dropdown menu item

Build a navbar where one of the items ("Products") opens a dropdown on hover using shadcn NavigationMenu. Include 4 sub-items with icon + title + description.
