Navegacion

Como los usuarios se mueven entre vistas y secciones de tu app. Navbar, sidebar, tabs, breadcrumbs, pagination, stepper, command palette, drawer, bottom nav y mega menu. Si no tenes claro cual usar cuando, el agente te va a devolver un navbar donde necesitabas un sidebar.

Como usar este modulo

Cada componente tiene demo real en Tailwind, anatomia, vocabulario cross-framework y prompts listos. La regla de oro: Navbar para apps publicas, Sidebar para dashboards, Tabs para contenido hermano, Breadcrumbs para jerarquia profunda. Lo demas es salsa.
