Tabs

Tabs Pestanas Segmented Control TabPanel

Alternar entre vistas relacionadas que comparten contexto. Uno activo a la vez. La anatomia Radix es la referencia universal: Root + List + Triggers + Contents. NO uses Tabs para navegar entre paginas, eso es Navbar. Tabs son para contenido que podria estar todo junto pero se divide por claridad.

Anatomia

Radix Tabs. Estandar.

Tabs.Root
Tabs.List (role="tablist")
Tabs.Trigger (value="a")

Tabs.Content (value="b")
  • Root Mantiene el estado del activo (value + onValueChange).
  • List Contenedor de triggers con role="tablist".
  • Trigger Botones clickeables. Uno por tab. Value = key.
  • Content Panel asociado a un value. Se muestra/oculta segun el activo.

Variantes

  • horizontal El default. Triggers en fila.
  • vertical Triggers en columna a la izquierda. Para documentacion.
  • pills Triggers con bg redondeado (como el demo de arriba).
  • underline Sin bg, solo border-bottom en el activo.
  • segmented Estilo iOS: grupo con bg + el activo tiene shadow.

Cuando usar y cuando no

  • Usalo cuando
  • Contenido relacionado que comparte contexto (settings: Account + Password).
  • 2-6 vistas alternativas.
  • El usuario va a saltar entre vistas.
  • NO lo uses cuando
  • Navegacion entre paginas distintas: usa Navbar/Sidebar.
  • Pasos secuenciales (wizard): usa Stepper.
  • 7+ tabs: el usuario se pierde, usa Sidebar.
  • Contenido no relacionado: usa secciones.

Prompts para coding agents

Settings page with 3 tabs

Create a shadcn Tabs component for a settings page with 3 tabs: Account, Password, Notifications. Each tab contains a form with relevant fields. Use TabsList, TabsTrigger and TabsContent.

Vertical tabs for documentation

Build vertical tabs with the trigger list on the left (width 200px) and content on the right. Use Radix Tabs with orientation="vertical". Include 5 sections: Intro, Installation, Usage, API, Examples.

Tabs with URL sync

Make the tabs component sync its active value with the URL query param ?tab=. On mount read the param, on change update it via Next.js router.replace. Use shadcn Tabs.

Underline-style tabs

Create tabs styled with no background, just a 2px border-bottom on the active trigger (border-foreground). Inactive triggers have text-muted-foreground and no border.
