Mega Menu

NavigationMenu Mega Menu Multi-column dropdown Mega Dropdown

Dropdown grande que se abre desde un navbar y contiene layout libre: columnas, cards, iconos, featured items, imagenes. Radix lo llama NavigationMenu y Shadcn lo porta con el mismo nombre. Lo usan masivamente ecommerce (Nike, Amazon), sitios con muchas categorias (Stripe docs) y paginas de producto complejas.

Demo interactiva

Navbar con un trigger "Products" que muestra un mega menu de 3 columnas.

Anatomia

Radix NavigationMenu. Content con layout libre adentro.

NavigationMenu.Root
NavigationMenu.List
NavigationMenu.Item
└ NavigationMenu.Trigger
└ NavigationMenu.Content (layout libre: grid/cards)
└ NavigationMenu.Link (multiples)
NavigationMenu.Viewport (rendered popover)
NavigationMenu.Indicator (arrow pointing trigger)
  • Root Wrapper que maneja estado de apertura.
  • List Lista horizontal de Items en la barra.
  • Item Un grupo Trigger + Content.
  • Trigger Boton en la nav que abre el content al hover/click.
  • Content El panel grande. Layout libre: grid, cards, featured.
  • Link Cada link adentro del content.
  • Viewport Donde se renderiza el content (evita layout shift).
  • Indicator Flechita que apunta al trigger activo.

Variantes

  • simple-dropdown Una columna de links (es el minimo).
  • multi-column 2-4 columnas con groups. El mega menu clasico.
  • with-featured Una columna es un card destacado (promo, blog post).
  • with-image Con imagen grande a un costado.
  • with-categories Hover en category a la izquierda muestra subitems a la derecha (amazon-style).

Cuando usar y cuando no

  • Usalo cuando
  • Ecommerce con muchas categorias.
  • Sitios corporativos con mucho contenido.
  • Queres destacar items especificos en el menu.
  • Necesitas agrupar links en secciones con contexto visual.
  • NO lo uses cuando
  • App simple con 5 links: usa Navbar plano.
  • Mobile-first: usa Drawer con acordeones.
  • Aplicaciones internas/dashboards: usa Sidebar.

Prompts para coding agents

Mega menu with 3 columns and featured card

Create a shadcn NavigationMenu with one trigger "Products" that opens a 3-column mega menu. Columns: Platform (Analytics, Engage), Developers (API Reference, SDK Docs), and a Featured card with a gradient background promoting "Introducing Acme 3.0". Each link has an icon, title and description.

Navigation menu with icon, title and description per item

Build a shadcn NavigationMenu where each NavigationMenuLink inside the content renders as a card with a lucide icon on the left, bold title and muted description. Use the ListItem pattern from shadcn docs.

Mega menu with hover delay

Create a NavigationMenu that opens on hover. Add a 150ms delay before showing and 300ms before hiding to avoid flicker. Use Radix delayDuration and skipDelayDuration props.

Ecommerce mega menu with category hover

Build an Amazon-style mega menu: left column lists categories, hovering one highlights it and shows its sub-items in the right column. Both columns are inside the NavigationMenuContent.
