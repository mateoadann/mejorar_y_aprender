Pagination

Pagination Pager Page navigator

Controles para moverse entre paginas de una lista larga con total conocido. El patron clasico: Prev + numeros de pagina + ellipsis + Next. Para scroll infinito o "load more" no es pagination: son otros patrones (Infinite Scroll, Load More button).

Anatomia

Shadcn Pagination: nav con list de items.

Pagination (nav)
PaginationContent (ul)
PaginationItem → PaginationPrevious
PaginationItem → PaginationLink
PaginationItem → PaginationEllipsis
PaginationItem → PaginationLink (active)
PaginationItem → PaginationNext
  • Pagination <nav aria-label="pagination">.
  • PaginationContent <ul> con los items en linea.
  • PaginationItem <li>. Wrapper generico.
  • PaginationLink Numero de pagina clickeable. isActive marca la actual.
  • PaginationPrevious / PaginationNext Links con icono + label.
  • PaginationEllipsis ... para truncar cuando hay muchas paginas.

Variantes

  • simple Solo Previous / Next (no numeros). Para blogs, articulos.
  • numbered Con numeros y ellipsis. El default.
  • with-page-size Select para elegir items por pagina (10, 25, 50).
  • with-total Muestra "Showing 1-10 of 234".
  • jump-to Input para saltar a una pagina especifica.
  • compact Solo prev/next + "4 / 10".

Cuando usar y cuando no

  • Usalo cuando
  • Lista larga con total conocido (tablas admin, search results).
  • Usuario necesita saltar a una pagina especifica.
  • SEO importa (cada pagina tiene URL propia).
  • Performance: no cargar todo junto.
  • NO lo uses cuando
  • Feed social o timeline: usa Infinite Scroll.
  • Listas cortas (<20 items): mostralos todos.
  • Total desconocido (cursor-based): usa "Load more".
  • Mobile: preferi infinite scroll o load more.

Prompts para coding agents

Pagination with page numbers and ellipsis

Create a shadcn Pagination component for 10 pages. Show Previous, page 1, ellipsis, pages around the current, ellipsis, last page, Next. Use the getPageRange utility pattern. Highlight the active page with isActive.

Simple prev/next pagination for blog posts

Build a minimal pagination with only Previous and Next buttons, plus a centered "Page X of Y" label. Use for a blog listing. Disable buttons at first/last page.

Pagination with page size selector

Create a pagination bar with: "Showing 1-10 of 234" on the left, page size Select (10/25/50/100) in the middle, and Pagination controls on the right. All synced via URL params page and pageSize.

Pagination synced to URL

Wire a shadcn Pagination to Next.js URL query param ?page=. Reading and updating via useSearchParams + router.push. The active page comes from the URL.
