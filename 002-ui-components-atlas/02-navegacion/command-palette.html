Command Palette

Command cmdk Command Menu ⌘K Quick Actions

Modal que se abre con ⌘K (o Ctrl+K) y te deja buscar, navegar y ejecutar acciones con el teclado. Lo popularizaron Linear, Raycast, Vercel y VS Code. Shadcn lo tiene como Command, basado en el hook cmdk de Paco Coursey. Es el componente de power-users por excelencia.

Anatomia

Shadcn Command es un wrapper sobre cmdk. Estructura clara.

CommandDialog (modal wrapper)
Command
CommandInput
CommandList
└ CommandEmpty
└ CommandGroup (heading)
└ CommandItem → CommandShortcut
└ CommandSeparator
  • CommandDialog El modal. Se abre con open + setOpen.
  • Command Root del cmdk. Maneja filtrado y teclado.
  • CommandInput Input con autofocus.
  • CommandList Contenedor scrolleable de items.
  • CommandEmpty Se muestra cuando no hay matches.
  • CommandGroup Seccion con heading.
  • CommandItem Una accion. onSelect con el value.
  • CommandShortcut Muestra atajo (⌘P) a la derecha del item.
  • CommandSeparator Divisor entre groups.

Variantes

  • standalone Embebido en la pagina (ej: search de docs).
  • dialog Modal con overlay y atajo global (el clasico).
  • with-recent Muestra recientes cuando el input esta vacio.
  • actions + navigation Combina "ir a" + "ejecutar accion" (Linear-style).
  • nested Entrar a un item abre otro nivel de opciones.

Cuando usar y cuando no

  • Usalo cuando
  • Apps con muchas acciones (Linear, Notion, Figma).
  • Power users que prefieren teclado sobre mouse.
  • Navegacion global + acciones en un solo lugar.
  • Necesitas acceso rapido desde cualquier pantalla.
  • NO lo uses cuando
  • Busqueda de contenido simple: usa Search Bar.
  • Seleccionar de una lista conocida: usa Select/Combobox.
  • Apps casuales donde nadie va a usar ⌘K.

Prompts para coding agents

Command palette with ⌘K global shortcut

Create a shadcn Command dialog that opens with ⌘K (or Ctrl+K). Use useEffect with keydown listener. Include 2 groups (Suggestions: Calendar, Search Emoji, Calculator and Settings: Profile, Billing, Settings) with keyboard shortcuts shown on the right via CommandShortcut.

Command palette with recent items and fuzzy search

Build a shadcn Command with a "Recent" group shown when input is empty. Use cmdk's built-in fuzzy search via the filter prop. Persist recent items in localStorage.

Command palette with navigation actions

Create a Command menu where each item is either a "Go to" action (navigates via Next.js router) or an "Action" (calls a function). Display a small badge next to each item indicating its type.

Nested command palette (pages)

Build a command palette where selecting certain items (like "Change theme") opens a nested level with sub-options (Light, Dark, System). Use a pages state array. Esc goes back one level.
