Breadcrumbs

Breadcrumb Path Trail Crumbs

Ruta jerarquica que muestra donde esta el usuario dentro de la app y le deja volver a niveles superiores. Util solo cuando la estructura es profunda (3+ niveles) y el usuario podria haber llegado desde links externos. HTML no tiene un elemento nativo: siempre es <nav aria-label="breadcrumb"> + <ol>.

Demo interactiva

Ruta tipica con chevron separators y el ultimo item como current (no clickable).

Anatomia

Shadcn Breadcrumb. Mapea 1:1 con HTML semantico.

  • Breadcrumb <nav aria-label="breadcrumb">.
  • BreadcrumbList <ol>. Lista ordenada porque el orden importa.
  • BreadcrumbItem <li>. Contiene un Link o un Page.
  • BreadcrumbLink Link clickable a un nivel superior.
  • BreadcrumbSeparator Chevron, slash u otro divisor.
  • BreadcrumbPage El ultimo item: no clickable, con aria-current="page".
  • BreadcrumbEllipsis Para truncar rutas largas.

Variantes

  • chevron Separador con ">" (el mas usado).
  • slash Separador con "/".
  • with-home-icon Primer item es un icono de casa en vez de texto.
  • truncated Con ellipsis en el medio cuando el path es largo: "Home / ... / Current".
  • dropdown-ellipsis El ellipsis abre un menu con los items ocultos.

Cuando usar y cuando no

  • Usalo cuando
  • Jerarquia profunda (3+ niveles).
  • Ecommerce, docs, file systems.
  • Usuarios pueden llegar por deep links.
  • Complementa al Sidebar mostrando "donde estoy".
  • NO lo uses cuando
  • Navegacion plana (2 niveles): usa Tabs o titulo simple.
  • Single-page apps sin jerarquia: es ruido visual.
  • Wizard steps: usa Stepper.

Prompts para coding agents

Breadcrumb with chevron separators and home icon

Create a shadcn Breadcrumb component. First item is a home icon (lucide Home) that links to "/". Use chevron-right as separators. Last item is the current page, non-clickable, bold. Path: Home / Products / Electronics / Laptop XPS 15.

Truncated breadcrumb with ellipsis dropdown

Build a breadcrumb that when the path has more than 4 segments, shows the first + ellipsis + last 2. The ellipsis is clickable and opens a DropdownMenu with the hidden items.

Auto-generated breadcrumb from URL

Generate breadcrumbs automatically from the current Next.js pathname. Split by "/" and create a BreadcrumbLink for each segment except the last. Capitalize segment names.

Breadcrumb with slash separators and no icons

Build a minimal breadcrumb with "/" as separator, no icons. Links use text-muted-foreground hover:text-foreground. Current page is text-foreground.
