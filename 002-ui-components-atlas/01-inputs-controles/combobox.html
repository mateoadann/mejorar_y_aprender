Combobox

Combobox Command Select Searchable Select cmdk

Select filtrable por texto. Compuesto en Shadcn con Popover + Command (cmdk). Ideal cuando tenes muchas opciones y el usuario necesita buscar. Puede ser single, multi-select con tags, o creatable.

Anatomia

Un Combobox real se arma componiendo Popover + Command.

Popover.Root
Popover.Trigger (Button)
Popover.Content > Command
├ CommandInput
├ CommandList
├ CommandEmpty
└ CommandGroup > CommandItem
  • Popover Abre y cierra el panel sobre el trigger.
  • Command Motor de filtrado (cmdk). Matchea por texto.
  • CommandInput El input que filtra en vivo.
  • CommandList Lista scrolleable de items.
  • CommandEmpty Mensaje cuando no hay resultados.
  • CommandGroup Agrupacion opcional con heading.

Cuando usar y cuando no

  • Usalo cuando
  • Tenes 20+ opciones y el usuario necesita filtrar.
  • La lista es conocida de antemano (no async).
  • Queres multi-select con tags.
  • Necesitas crear items on-the-fly (creatable).
  • NO lo uses cuando
  • Son pocas opciones: usa Select o Radio Group.
  • Datos vienen de API: considera Autocomplete async.
  • Es una accion global: usa Command Palette (modal).

Prompts para coding agents

Searchable combobox for framework selection

Build a shadcn Combobox composed of Popover + Command for selecting a frontend framework. Use CommandInput for filtering, CommandGroup with heading "Frontend", and CommandItem for each framework. Show a checkmark next to the selected one.

Multi-select combobox with tags

Create a multi-select combobox. Selected values appear as removable tags inside the trigger. Clicking an item toggles it. Show up to 3 tags inline, then "+N more". Use shadcn Popover + Command.

Creatable combobox for tags

Build a creatable combobox: if the typed query does not match any existing item, show a "Create [query]" item at the bottom. Clicking it adds the new value to the list.
