Search Bar

SearchBarSearchInputOmniSearch

Input de busqueda inline con icono de lupa a la izquierda y clear button a la derecha cuando hay texto. OJO: NO es lo mismo que Command Palette. Search Bar vive inline en un header o toolbar; Command Palette es un modal global que abris con Cmd+K.

Cuando usar y cuando no

  • Usalo cuando
  • Busqueda de contenido inline (header, toolbar, tabla).
  • Filtrar una lista visible en la pagina.
  • Busqueda global con submit a una pagina de resultados.
  • NO lo uses cuando
  • Es una navegacion global con acciones: usa Command Palette.
  • El usuario elige de una lista fija: usa Select/Combobox.
  • Es un form field de texto: usa Input normal.

Prompts para coding agents

Search bar with debounced input and clear button

Build a search bar using shadcn Input with a leading search icon and a trailing clear button (X). Debounce onChange by 300ms and call onSearch(query). The clear button appears only when value is non-empty.

Command palette with Cmd+K shortcut

Build a global Command Palette using shadcn Dialog + Command (cmdk). Open with Cmd/Ctrl+K. Groups: "Suggestions", "Navigation", "Actions". Each item shows an icon and label, keyboard navigable.

Table filter search bar

Add a search bar above a data table that filters rows by column [COLUMN_NAME] in real-time. Use shadcn Input with search icon. Debounce 200ms.
