Cuando usar y cuando no

  • Usalo cuando
  • Subir documentos, imagenes, attachments.
  • Queres drag & drop.
  • Necesitas mostrar progress de subida.
  • Multi-file con limite.
  • NO lo uses cuando
  • Basta con la camara: usa capture attr.
  • Es un avatar circular: usa Avatar Uploader especializado.
  • Import masivo: considera CSV importer con preview.

Prompts para coding agents

Image dropzone with preview and upload progress

Build an image upload component using react-dropzone. Accept only image/*. Show thumbnails in a grid. Each thumbnail displays a progress bar while uploading. Use PUT to [UPLOAD_URL].

Multi-file upload with max 5 files

Create a FileUpload with max 5 files, max 10 MB each, PDF only. Show a list with name, size and remove button. Disable dropzone when limit is reached. Use shadcn styling.

Avatar uploader with crop

Build a round avatar uploader: clicking opens file picker, after select show a modal with a square crop tool, save cropped image to [ENDPOINT].
