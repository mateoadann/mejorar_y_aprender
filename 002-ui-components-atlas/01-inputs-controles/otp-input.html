OTP Input

InputOTPPinInputVerificationCode

Input especializado para codigos de verificacion: una celda por digito, focus automatico al tipear, paste que rellena todas las celdas de una. Para 2FA, SMS, email verification, PIN.

Cuando usar y cuando no

  • Usalo cuando
  • Verificacion por SMS, email o authenticator app.
  • PIN de acceso de 4-6 digitos.
  • Queres feedback visual de progreso del codigo.
  • NO lo uses cuando
  • El codigo es muy largo (10+): usa Input normal.
  • Es una contraseña: usa Input password.
  • El codigo alfanumerico libre: usa Input.

Prompts para coding agents

6-digit OTP with auto-submit on complete

Build a 6-digit OTP input using shadcn InputOTP. Auto-focus moves to the next slot on input and back on backspace. When all 6 digits are entered, auto-submit the form. Accept paste that fills all slots at once.

OTP with 3-3 groups separated by a dash

Create an OTP input of 6 digits split into two groups of 3 with a dash separator. Use shadcn InputOTP with InputOTPSeparator between InputOTPGroups.

OTP with resend timer

Build a 6-digit OTP input with a "Resend code" link below that becomes clickable after 30s. Show the countdown while disabled. Call POST /api/otp/resend on click.
