Cuando usar y cuando no

  • Usalo cuando
  • El input requiere mas de una linea de texto.
  • Comentarios, descripciones, mensajes, feedback.
  • Necesitas un counter o un limite suave.
  • El contenido es plano (sin formato rich text).
  • NO lo uses cuando
  • Solo necesitas una linea: usa Input.
  • Queres formato (bold, listas): usa un Rich Text Editor.
  • Es codigo: usa un Code Editor tipo Monaco.
  • Es un campo numerico: usa Input type="number".

Comment textarea with markdown preview

Textarea with validation error

Create a textarea with react-hook-form validation. Require min 10 chars, max 500. Show destructive border and error text below when invalid. Label: "[LABEL]".

Textarea with submit on Cmd+Enter
