Slider

SliderRangerange input

Input continuo o discreto dentro de un rango numerico. Se arma con un track, un range (el tramo pintado) y uno o dos thumbs. Para volumen, brillo, precio min/max, rating con numero.

Cuando usar y cuando no

  • Usalo cuando
  • Valor en un rango continuo (volumen, brillo, zoom).
  • El usuario tiene una intuicion visual del valor.
  • Rangos min/max (precio, fechas).
  • Puntos discretos con steps claros.
  • NO lo uses cuando
  • Necesitas precision exacta: usa Input number.
  • Son pocas opciones discretas: usa Radio o Select.
  • El valor no tiene orden: usa ToggleGroup.
