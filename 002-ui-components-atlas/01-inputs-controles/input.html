Input

Input TextField <input> FormField

Campo de una sola línea para texto, email, password, number, tel, url. Es el componente más aburrido del mundo y el más importante: si lo hacés mal, el 90% de tu form está mal.

Demo interactiva

Escribí, tabuleá entre los campos, mirá los estados.

Te mandaremos un link de verificación.

El username debe tener al menos 3 caracteres.

Anatomia

Un campo de formulario accesible tiene 4 partes.

FormField (wrapper)
FormLabel — "Email"
Input.Root
[icon]
<input>
[adornment]
FormDescription — helper text
FormMessage — error text
  • Label Siempre asociado con htmlFor + id.
  • Input.Root Puede tener adornments (iconos) leading o trailing.
  • Description Texto helper debajo (opcional).
  • Message Error de validación. Solo visible cuando hay error.

Cuando usar y cuando no

  • Usalo cuando
  • Necesitás capturar texto corto (< 100 caracteres)
  • El valor es email, password, número, teléfono, URL
  • Hay validación inline
  • Es parte de un formulario típico
  • NO lo uses cuando
  • El texto es largo o multilínea — usá Textarea
  • Es un código de N dígitos — usá OTP Input
  • El usuario elige de una lista — usá Select o Combobox
  • Es una fecha — usá Date Picker

Prompts para coding agents

Email input with inline validation

Create a shadcn Input with type="email", a Label "Email", and inline validation using react-hook-form + zod. Show a FormMessage error when the email is invalid. Placeholder: "name@example.com". Include a helper text "We'll send a verification link".

Password with show/hide

Create a password Input with a show/hide toggle button (Eye / EyeOff lucide icons) positioned as a trailing adornment. Clicking the icon toggles the input type between "password" and "text". Include aria-label on the toggle button.

Input with leading icon

Create a search Input with a Search lucide icon as a leading adornment. Use pl-9 to leave space for the icon. The input should be inside a relative wrapper; the icon is absolutely positioned.

Number input with stepper

Create a number Input with min=0, max=100, step=1. Add two buttons (+ and -) as trailing adornments that increment/decrement the value. Use a React state for the value.
