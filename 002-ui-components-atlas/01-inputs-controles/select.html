Select

Select Dropdown Picker Listbox

Dropdown para elegir UNA opcion de una lista fija, conocida de antemano. Si el usuario tiene que tipear para filtrar, no es Select: es Combobox. La anatomia de Radix es el estandar de la industria.

  • Trigger Boton que abre el menu. Muestra el Value actual.
  • Portal Monta el Content fuera del flujo (evita overflow clipping).
  • Content El popover con las opciones.
  • Viewport Area scrolleable del content.
  • Group + Label Agrupa items bajo un header.
  • Item Una opcion. Contiene ItemText + ItemIndicator (check).
  • Separator Linea divisoria entre groups.

Cuando usar y cuando no

  • Usalo cuando
  • Tenes entre 5 y 20 opciones conocidas.
  • El usuario elige UNA sola opcion.
  • No necesitas filtrar/buscar dentro de la lista.
  • La lista no cambia dinamicamente.
  • NO lo uses cuando
  • Son 2-4 opciones: usa Radio Group.
  • Son 100+ opciones: usa Combobox.
  • El usuario necesita tipear: usa Combobox.
  • Eligen varias: usa MultiSelect o Checkbox group.

Prompts para coding agents

Country select with flags

Create a shadcn/ui Select for country selection. Each item has a flag emoji on the left and country name. Group items by continent using Select.Group + Select.Label. Include a Separator between groups.

Select grouped by category

Build a Select with items grouped into [CATEGORY_A] and [CATEGORY_B]. Use Radix Select.Group, Select.Label, and Select.Separator. The currently selected item should show a checkmark via Select.ItemIndicator.

Select with form validation

Wire up a shadcn Select to react-hook-form. Require a value, show destructive border and error message below on invalid submit. Label: "[LABEL]".
