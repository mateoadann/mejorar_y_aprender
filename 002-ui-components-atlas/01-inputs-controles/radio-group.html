Radio Group

RadioGroupradiooption

Elegir UNA sola opcion de varias. Se muestra todo al mismo tiempo (a diferencia de Select), lo cual es mas rapido para 2-5 opciones. Nunca uses radio individual: siempre en grupo.

Demo interactiva

Metodo de envio

Anatomia

RadioGroup.Root
RadioGroup.Item > RadioGroup.Indicator
+ <label>
RadioGroup.Item > RadioGroup.Indicator
+ <label>
  • Root Contenedor del grupo. Maneja el value seleccionado.
  • Item Cada opcion (un radio individual).
  • Indicator El puntito interior cuando esta seleccionado.
  • Label Texto asociado a cada item.

Cuando usar y cuando no

  • Usalo cuando
  • El usuario elige UNA opcion de varias.
  • Son 2-5 opciones que caben visibles.
  • Queres que se vean todas las opciones al mismo tiempo.
  • Payment method, shipping option, size.
  • NO lo uses cuando
  • Son 6+ opciones: usa Select.
  • Se pueden elegir varias: usa Checkbox.
  • Es un toggle on/off: usa Switch.
