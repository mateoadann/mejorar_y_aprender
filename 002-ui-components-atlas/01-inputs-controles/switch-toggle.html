Switch / Toggle

SwitchToggleToggleGroupiOS toggle

Tres primos que la gente confunde: Switch (iOS toggle, para settings persistidos), Toggle (boton que queda presionado, tipo bold) y ToggleGroup (grupo de toggles exclusivos o inclusivos, tipo alineacion de texto).

Cuando usar y cuando no

  • Usalo cuando
  • Switch: setting que se aplica al toque (dark mode, notificaciones).
  • Toggle: boton que recuerda su estado (bold, italic).
  • ToggleGroup: seleccion exclusiva visual (align left/center/right).
  • NO lo uses cuando
  • El cambio necesita "Save": usa Checkbox en un form.
  • Ejecuta una accion one-shot: usa Button.
  • El usuario elige entre muchas: usa Select o RadioGroup.

Dark mode switch with icons

Build a dark mode Switch using shadcn. Show a sun icon when off and moon icon when on, animated inside the thumb. Persist to localStorage and toggle the .dark class on the html element.

Toggle group for text formatting

Create a ToggleGroup type="multiple" with three items: Bold, Italic, Underline. Each item is an icon button. Controlled state, emits onValueChange with the active array.

Settings page with multiple switches

Build a settings section with 4 shadcn Switch rows: Email notifications, Push, SMS, Weekly digest. Each row has label left, description below, switch right. No save button, changes apply instantly.
