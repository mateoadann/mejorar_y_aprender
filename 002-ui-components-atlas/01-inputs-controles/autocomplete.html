Autocomplete

AutocompleteTypeaheadSuggest

Input libre con sugerencias mientras el usuario tipea. La diferencia clave con Combobox: Autocomplete suele aceptar valores que no estan en la lista (freeSolo), y a menudo carga las sugerencias async desde una API.

Cuando usar y cuando no

  • Usalo cuando
  • Las sugerencias vienen de una API.
  • El usuario puede ingresar valores que no estan en la lista.
  • Buscar tags, usuarios, direcciones, ciudades.
  • El dataset es grande o infinito.
  • NO lo uses cuando
  • La lista es chica y conocida: usa Select.
  • El usuario solo busca contenido: usa Search Bar.
  • La lista es local y fija: usa Combobox.

Prompts para coding agents

Async autocomplete fetching users from API

Build an autocomplete input that debounces keystrokes by 300ms and fetches users from GET /api/users?q=[query]. Show a spinner while loading, "No results" when empty, and render each user with avatar + name.

Tag autocomplete that allows creating new tags

Create an autocomplete for tags. freeSolo is enabled so users can type new values and press Enter to add them. Show selected tags as chips above the input. Use MUI Autocomplete or shadcn Combobox with freeSolo behavior.

Address autocomplete with Google Places

Build an address autocomplete using Google Places Autocomplete API. Suggestions include street, city, country. On select, populate the full address object into the form.
