Cuando usar y cuando no

  • Usalo cuando
  • Elegir una fecha o rango de fechas.
  • Booking, filtros de reportes, due date.
  • Necesitas deshabilitar ciertos dias.
  • Queres mostrar presets (Hoy, Ayer, Ultimos 7).
  • NO lo uses cuando
  • Solo necesitas hora: usa Time Picker.
  • La fecha es aproximada: usa Select con "Este mes / mes pasado".
  • Es fecha muy lejana (nacimiento): usa 3 selects (dia/mes/año).

Prompts para coding agents

Date range picker with presets

Build a shadcn DatePicker with range mode and a preset sidebar: Today, Yesterday, Last 7 days, Last 30 days, This month. Clicking a preset sets both dates and closes the popover.

Booking date picker that disables past dates

Create a booking DatePicker using react-day-picker. Disable all dates before today. Highlight today with a ring. Format the trigger button as "DD MMM YYYY".

DatePicker wired to react-hook-form

Wire a shadcn DatePicker to react-hook-form. Required field. Validate that the date is not in the past. Show destructive border and error message.
