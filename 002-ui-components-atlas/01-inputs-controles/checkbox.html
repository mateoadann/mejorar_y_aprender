Checkbox

Checkboxtickboxcheck

Toggle binario con un tercer estado: indeterminate. Se usa para select-all en tablas, configuraciones multi-opcion y terms & conditions. Siempre va con un label clickeable.

Demo interactiva

Select-all con estado indeterminate en vivo.

Anatomia

Checkbox.Root (button)
Checkbox.Indicator (icono check/minus)
+ <label> asociado via htmlFor
  • Root El boton clickeable con el estado.
  • Indicator El icono visible (check o minus).
  • Label Texto clickeable asociado al input.

Cuando usar y cuando no

  • Usalo cuando
  • Opcion binaria independiente (on/off).
  • Multiples opciones donde cada una es independiente.
  • Select-all en tablas o listas.
  • Terms & conditions, consentimientos.
  • NO lo uses cuando
  • Opcion unica de varias: usa Radio Group.
  • Setting persistido tipo dark mode: usa Switch.
  • Toggle de formato (bold, italic): usa Toggle.

Prompts para coding agents

Checkbox with indeterminate state for select-all

Build a data table header checkbox that controls all row checkboxes. When no rows are selected it is unchecked, all rows checked is checked, and some rows checked is indeterminate. Use shadcn Checkbox and sync via state.

Terms and conditions checkbox with link

Create a terms checkbox where the label contains a link "I accept the [Terms of Service]". Disable the submit button until the checkbox is checked. Show a destructive error if user submits unchecked.

Checkbox group with react-hook-form

Build a checkbox group for "Notification preferences" with 4 options. Use react-hook-form useFieldArray or a checkbox array field. Require at least one option selected.
