Button

Button btn <button> IconButton LoadingButton

El disparador de acciones. Sirve para decirle al sistema "hacé esto ahora": submit, save, delete, open modal. Si el click lleva a una URL, no es Button: es Link. Entenderlo evita el 80% de los errores de accesibilidad.

Demo interactiva

Clickealos. Probá hover, focus con Tab, disabled.

Anatomia

El button es plano, pero tiene partes nombradas.

Button.Root <button>
Icon leading
Label
Icon trailing / Spinner
  • Root El elemento <button> nativo. Define type="button|submit|reset".
  • Icon leading Ícono antes del label. Opcional.
  • Label El texto de la acción. Empezá con verbo: "Save", "Delete".
  • Icon trailing Ícono después del label. Flechas, chevrons.
  • Spinner Reemplaza el leading icon cuando está loading.

Cuando usar y cuando no

  • Usalo cuando
  • La acción cambia el estado de la app (save, delete, submit)
  • Abre un modal, dialog o popover
  • Dispara un request (login, fetch)
  • Activa un comportamiento local (toggle dark mode, play video)
  • NO lo uses cuando
  • El click navega a otra URL — usá <a> o Link
  • Es solo texto decorativo — usá span o label
  • Es un toggle con dos estados persistentes — usá Switch
  • Son varias opciones exclusivas — usá RadioGroup o ToggleGroup

Primary button with loading state

Add a primary Button from shadcn/ui with a loading state. When clicked, it disables itself, shows a spinner to the left of the label, and changes the label from "Save" to "Saving...". Use the `disabled` prop bound to a `isLoading` state. Label: "Save".

Destructive button with confirmation

Create a destructive Button (variant="destructive") labeled "Delete account". On click, open a shadcn AlertDialog asking "Are you absolutely sure?" with Cancel and Continue. Continue triggers the destructive action.

Icon-only button with tooltip

Create an icon-only Button (size="icon") with a Settings lucide-react icon. Wrap it in a shadcn Tooltip with content "Open settings". Include `aria-label="Open settings"` on the button for accessibility.

Button as link (asChild)

Create a Button variant="outline" that navigates to "/dashboard". Use the `asChild` pattern to wrap a Next.js Link component so the semantics are correct (anchor, not button). Label: "Go to dashboard".

Button group (toolbar)

Create a horizontal button group with three shadcn Buttons: "Bold", "Italic", "Underline". Use `variant="outline"` and remove the gap between them so the inner borders collapse. Wrap them in a div with `role="group"` and `aria-label="Text formatting"`.
