Rating

RatingStarsScore

Input discreto con una escala visual (tipicamente estrellas). Puede ser interactivo (el usuario califica) o read-only (mostrar un promedio). Soporta half-step e iconos custom.

Cuando usar y cuando no

  • Usalo cuando
  • Reviews y ratings de productos.
  • Feedback de satisfaccion visual.
  • Mostrar promedios de puntaje.
  • Favoritos con corazon.
  • NO lo uses cuando
  • Necesitas feedback textual: agrega un textarea.
  • El rating es continuo (0-100): usa Slider.
  • Es un favorite binario: usa Toggle con heart icon.

Prompts para coding agents

5-star rating with half-star precision

Build a 5-star rating component using MUI Rating or a custom shadcn-styled version. Allow half-star precision (step 0.5). Display the numeric value next to the stars.

Read-only rating from API average

Create a read-only Rating component that receives a number (e.g. 4.3) and renders filled, half and empty stars accordingly. Show "(4.3)" next to it.

Rating with tooltip labels

Build an interactive rating where hovering each star shows a tooltip with label: "Terrible", "Bad", "OK", "Good", "Excellent". On click, submit via onChange.
