Tooltip

Tooltip title Hint Popup label

Hint de texto corto que aparece al hover o focus de un elemento. Es el pattern para explicar icon buttons ("¿qué hace este botón de engranaje?"), mostrar keyboard shortcuts, o dar contexto breve. Jamás interactivo: si el contenido necesita clicks, es Popover.

Tooltip ≠ Popover: tooltip aparece al hover/focus, es pasivo, texto simple. Popover aparece al click, tiene contenido rico, es interactivo. Y en mobile los tooltips NO existen — no hay hover en touch.

Anatomía

Provider global + Root + Trigger + Content.

Tooltip.Provider (delayDuration global, una vez en el root)
Tooltip.Root
Tooltip.Trigger (el elemento con hover)
Tooltip.Portal → Tooltip.Content (+ Arrow)
  • Provider Se monta una vez. Controla delays globales.
  • Root Un wrapper por tooltip. Maneja open state.
  • Trigger El elemento al que se ancla. Usá asChild.
  • Content El bubble con el texto. role="tooltip" automático.
  • Arrow Triangulito apuntando al trigger (opcional).

Cuándo usar y cuándo no

  • Usalo cuando
  • Label para icon-only buttons
  • Mostrar keyboard shortcuts (Save ⌘S)
  • Explicar labels truncados (texto cortado con ...)
  • Hints cortos en form fields ayuda/pista
  • NO lo uses cuando
  • Contenido interactivo → Popover
  • Info crítica que el usuario debe ver siempre
  • Mobile-only: no hay hover
  • Texto largo → Popover o un nuevo pattern

Prompts para coding agents

Tooltip on icon button with shortcut

Create a shadcn Button size="icon" with a Trash2 lucide icon wrapped in a Tooltip. The tooltip content should read "Delete" followed by a <kbd>⌘⌫</kbd>. Add aria-label="Delete" to the button.

Tooltip with delay for help text

Add a shadcn Tooltip to a form field label with delayDuration={700}. The tooltip explains the field purpose: "Your full legal name as it appears on your ID." Use a small Info lucide icon as the trigger next to the label.

Tooltip for truncated text

When a table cell has truncated text (text-overflow: ellipsis), wrap it in a shadcn Tooltip that only shows when the text is actually truncated. Use a ref and check scrollWidth > clientWidth.

Tooltip provider setup

Wrap the app root with <TooltipProvider delayDuration={300} skipDelayDuration={500}> so all tooltips share consistent timing. Radix re-exports TooltipProvider from @radix-ui/react-tooltip.
