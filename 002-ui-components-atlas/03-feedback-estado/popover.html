Popover

Popover Popover.Content FloatingPanel Overlay panel

Panel flotante anclado a un trigger que muestra contenido rico (forms, previews, pickers). Abre on-click (no hover como tooltip), no bloquea la UI como un Dialog, y se cierra clickeando afuera o con ESC. Es el pattern que usan color pickers, date pickers inline y quick actions.

Popover vs Tooltip vs Dialog: Tooltip = texto corto, hover, no interactivo. Popover = contenido rico, click, interactivo. Dialog = bloquea toda la UI, centrado.

Cuándo usar y cuándo no

  • Usalo cuando
  • Mini-forms rápidos (rename, edit cell)
  • Color pickers, date pickers, emoji pickers
  • User profile preview on click
  • Quick actions contextual a un elemento
  • NO lo uses cuando
  • Texto corto descriptivo → Tooltip
  • Flujo que bloquea la UI → Dialog
  • Menu de opciones → DropdownMenu
  • Mobile: priorizá Sheet o pantalla dedicada

Prompts para coding agents

Popover with quick edit form

Create a shadcn Popover triggered by a "Edit" icon button on a table row. Inside the content, render a small form with a Name input and a Save button. On save, call the update mutation and close the popover.

User profile preview on click

Wrap every @username mention with a Popover that opens on click and shows the user avatar, full name, bio, and a "Follow" button. Use Popover.Trigger asChild with the mention anchor element.

Color picker popover

Create a Popover with a trigger that shows the current selected color. Inside, render a grid of 12 preset colors plus a native color input. Clicking a color updates the parent state and closes the popover.

Filter panel popover

Create a "Filters" button that opens a Popover with multiple checkboxes (Status, Type, Assignee). On change, update the URL search params. The popover stays open while filtering.
