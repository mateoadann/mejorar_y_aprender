Banner

Banner Announcement Notice Bar Cookie Banner TopBar

Barra full-width en el top de la página que comunica algo a nivel app: cookie notice, programmed maintenance, promo global, "estás en modo preview". No pertenece a una sección específica: es contexto para toda la sesión.

Banner vs Alert: Banner es full-width, pegado al borde superior, afecta a toda la app. Alert vive dentro del contenido de una página. Banner normalmente tiene dismiss persistente en localStorage.

Anatomía

Root full-width + Icon + Message + [Action] + [Close].

  • Root Div full-width, típicamente con border-b. Sticky opcional.
  • Icon Info, warning, megaphone (para promos).
  • Message Un solo line, más corto que un Alert.
  • Action Botón o link con la CTA (Read more, Accept).
  • Close Persiste en localStorage para no reaparecer.

Cuándo usar y cuándo no

  • Usalo cuando
  • Cookie consent / GDPR notice
  • Anuncio de mantenimiento programado
  • Promos o announcements a nivel app
  • "Estás en modo preview / staging / impersonation"
  • NO lo uses cuando
  • Info contextual a una página → Alert
  • Mensaje temporal → Toast
  • Error crítico bloqueante → Dialog
  • Abusar de banners apilados — el usuario los ignora

Prompts para coding agents

Cookie consent banner

Create a sticky top Banner for cookie consent. It has a message, an Accept button, a Decline button, and an X to dismiss. Persist the user choice in localStorage under key "cookie-consent". If the key exists, do not render the banner.

Maintenance warning banner

Create a warning Banner with an AlertTriangle lucide icon reading "Scheduled maintenance: Sunday 2am-4am UTC." Include a "Read more" button that opens a Dialog with details. The banner is full-width, yellow tint, dismissible.

Impersonation warning banner

When an admin is impersonating a user, render a red Banner at the top reading "You are viewing as {userName}" with a "Stop impersonating" button that calls a mutation. The banner is sticky and cannot be dismissed.

Promo banner with gradient

Create a promotional Banner with a gradient background (purple to blue) and a megaphone icon reading "Black Friday: 50% off Pro plans". Include an "Upgrade" button that links to /pricing. Dismissible for 24h via localStorage with timestamp.
