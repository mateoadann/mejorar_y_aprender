¿Qué componente usar?

Un árbol de decisión visual para cuando sabés el problema pero no sabés el nombre del componente. Arrancá por el grupo que matchea tu caso y seguí las ramas hasta el componente.

1¿Necesito que el usuario elija entre opciones?

¿Cuántas opciones hay?

2-4 opciones mutuamente excluyentes Radio Group
5-15 opciones, una elegida Select
15+ con búsqueda Combobox
Múltiples seleccionables a la vez Checkbox Group

¿Es un toggle binario de estado?

Estado persistido (dark mode, notifications) Switch
Botón presionable (bold/italic en un editor) Toggle

2¿Cómo mostrar información al usuario?

¿Es transitoria (aparece y desaparece sola)?

Confirmación de una acción que el usuario hizo Toast

¿Es contextual inline (no bloquea)?

Alerta dentro del flujo de la página Alert

¿Es page-level (afecta a toda la app)?

Cookies, mantenimiento, promos Banner

¿Es crítica y bloqueante?

Confirmación destructiva ("¿eliminar?") Alert Dialog
Formulario o acción contextual Dialog
Imagen, video o contenido inmersivo Modal Fullscreen

3¿Cómo estructurar datos en pantalla?

¿Son datos tabulares?

Simple, read-only Table
Con sort, filter y pagination Data Table

¿Son listas de items?

Items homogéneos (mismo tipo) List
Items heterogéneos con imágenes y CTAs Card Grid
Jerárquicos (archivos, categorías) Tree View
Cronológicos (activity, historial) Timeline

¿Son métricas o KPIs?

Dashboard con números grandes Stat Card

¿Son tasks con múltiples estados?

To do / Doing / Done Kanban Board

4¿Cómo navegar la app?

¿Navegación principal desktop?

Pocas secciones (3-6) Navbar
Muchas secciones o dashboard Sidebar

¿Navegación principal mobile?

3-5 secciones principales Bottom Nav
Muchas secciones, oculto hasta abrir Drawer

¿Contenido relacionado en la misma página?

Mismo contexto, vistas alternativas Tabs
Páginas jerárquicas (ruta actual) Breadcrumbs

¿Listas largas?

Total de items conocido Pagination
Feed continuo tipo social Infinite Scroll

¿Pasos secuenciales?

Formulario largo partido en pasos Form Wizard
Proceso visual (onboarding) Stepper

5¿Cómo mostrar loading?

¿Duración conocida?

Porcentaje exacto de avance Progress

¿Duración desconocida?

Acción puntual (save, submit) Spinner
Contenido cargando (lista, página) Skeleton

6¿Cómo mostrar info contextual flotante?

¿Texto simple?

Hover de un label o botón icon-only Tooltip

¿Contenido rico o interactivo?

Hover con info previa (perfil, link preview) Hover Card
Click con interacción (form inline) Popover
Menú de acciones disparado por botón Dropdown Menu
Menú disparado por click derecho Context Menu
