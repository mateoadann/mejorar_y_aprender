¿Cómo usar este módulo?

Paso 1

Sabés el nombre

Andá a la Tabla Global y tipealo. Filtra por alias también ("dropdown", "modal", "loader").

Paso 2

Sabés el problema

Tenés un problema pero no sabés el nombre del componente. Usá el Decision Tree.

Paso 3

Querés generarlo

Ya elegiste el componente. Copiá un prompt de la Prompt Library y pegalo en tu coding agent.
